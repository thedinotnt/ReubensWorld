This Website

This website was made by hand in html code, and hosted on an AWS (Amazon Website Services) and was up and running, behind a password, on the 19th July 2024.

I was 13 years old when I made this site and I set it up all by myself.

I used VS code's live preview to check linked to github, which is linked to the AWS Amplify server that redeploys the server every update I commit.

When I find ways of stuff working, I dont change it, so rather than it being 1 line; I'll just make it 50 if thats how I know how to do it.

At first I wanted the domain Reuben.fun, but then godaddy's recommendations showed me a list of 40 more, so then I wanted Reubens.world and thats what im sticking to now
